ZanobiJS CLI es una herramienta de interfaz de línea de comandos que le ayuda a iniciar,
desarrollar y mantener las aplicaciones.

### Instalación

```sh
npm install -g @zanobijs/cli
```

### Ayuda

```sh
zanobijs --help
```

### Descripción general de los comandos

<table>
  <tr>
    <td>Comando</td>
    <td>Alias</td>
    <td>Descripción</td>
  </tr>
  <tr>
    <td>`new`</td>
    <td>n</td>
    <td>Crea una nueva aplicación estándar</td>
  </tr>
  <tr>
    <td>`generate`</td>
    <td>g</td>
    <td>Genera entidades</td>
  </tr>
  <tr>
    <td>`build`</td>
    <td></td>
    <td>Compila una aplicación en una carpeta de salida</td>
  </tr>
</table>
