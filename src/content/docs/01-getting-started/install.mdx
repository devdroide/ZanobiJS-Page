### Requisitos

- NodeJS (version >= 16) debe estar instalado en su máquina.
- Editor de código - Recomendamos VS Code.
- Terminal - ZanobiJS es usado a través de la interfaz de línea de comandos (CLI).

### Instalación

##### 1. Descarga el CLI de ZanobiJS

Ingresa a la terminal y coloca el siguiente comando:

```sh
npm install -g @zanobijs/cli
```

##### 2. Crear una aplicación

Ahora vamos a crear una aplicación:

```sh
zanobijs new nombre_app
```

:::tip[¿Sabías esto?]
Puedes usar de forma abreviada el comando [new] con [n] simplemente
:::

Si todo sale bien en tu terminal podrás ver "😎 SUCCESS - The application nombre_app was created correctly"

##### 3. Ingresando a la aplicación

Vamos ingresar a la carpeta de la aplicación e instalar sus dependencias:

```sh
cd nombre_app
npm install
```
