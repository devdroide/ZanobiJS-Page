Los controladores son los encargados de aplicar la lógica necesaria a la petición que ingresa
y a la respuesta que se entrega.

<Image src={imgControllerEsDark} alt="App-Controller-Es Dark" />

:::tip[¿Sabías esto?]
No debes aplicar la lógica de negocio aquí, eso va en los servicios 😉
:::

```js title="Controlador de ejemplo"
import { Controller } from "@zanobijs/common";
import { ServiceExample } from "./example.service";

@Controller()
export class ControllerExample {
  constructor(
    private sExample: ServiceExample,
  ) {}

  getHelloService() {
    return this.sExample.getHello();
  }
}
```
